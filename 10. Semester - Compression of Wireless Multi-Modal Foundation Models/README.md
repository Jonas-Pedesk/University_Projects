# Wireless Multi-Modal Foundation Model Post Training Compression | Feb. 2026 – June 2026
This project investigates the effects of applying compression techniques to a transformer architecture neural network machine learning model. A documentation is made for the effects of Pruning, Quantization and Knowledge distillation in a simulated scenario where random generated QAM points would be transmitted by a Rician channel to simulate a multipath effect in a factory building. All simulations are evaluated by different performance metrics which serve as indications of how great the level of compression can be in both scenarios where individual and combined compression methods are applied. After adjusting the compression methods, it is shown that both a Quantize-Prune-KD and a Prune-Quantize-KD pipeline are potential candidates for future Foundation Models, since they both showed reduction in matters of memory and parameter count while retaining accuracy.
