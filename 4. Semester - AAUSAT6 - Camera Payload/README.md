# AAUSAT6 – Camera Payload | Feb. 2023 – July 2023
This project examines the process of designing an embedded system for the AAUSAT platform. Specifically the project involves the design of a camera imaging payload prototype. The prototype is implemented with a CMOD FPGA handling image extraction from the camera over UART to a STM32 MCU. The MCU performs image compression and storage handling of captured image data. The MCU uses FreeRTOS kernel for synchronization and scheduling of the embedded design. The prototype employs several communication protocols including CAN, which can be used to initiate an image capture request at any given UTC time slot. The compressed data is accessible through a SD card written to over an SPI bus. The current prototype complies with set requirement relating to image handling, compression, and storage.
