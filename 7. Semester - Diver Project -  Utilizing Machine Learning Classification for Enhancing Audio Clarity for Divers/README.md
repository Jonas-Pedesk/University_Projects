# Voice and Noise Activity Detection for Professional Divers | Sep. 2024 – Dec. 2024 
This project sets up a machine learning model for classifying different parts of a diver’s audio signal at Nordsøen Oceanarium. The purpose of the model is to see if the audio levels of the characteristics of the voice of the diver can be controlled separately from the undesired sounds, as this may improve the experience of the audience. Audio recordings of the diver were explored. After having the audio labeled, high classification accuracies could be achieved across different machine learning models. It was possible to use the predictions to segment audio files based on four categories, but final tests showed that varying levels of unintended information loss occurred when attenuating the audio.
