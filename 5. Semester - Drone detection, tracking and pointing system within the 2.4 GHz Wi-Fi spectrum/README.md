# Drone Detection, Tracking, and Pointing in the 2.4 GHz Wi-Fi Band | Sep. 2023 – Dec. 2023
The amount of violations in restricted airspaces is increasing as drones are becoming more accessible. This project aims to study for and develop a prototype system, able to detect, track and point at a wifi-capable drone. The design to achieve this is through detecting RF-signals using Angle of Arrival (AoA). The prototype is designed to calculate the AoA from a 2.44 GHz centre frequency signal received through an antenna array consisting of patch antennas. A micro strip bandpass filter and a digital filter is designed in order to attenuate unwanted frequencies before sampling the signal using an SDR. The AoA is calculated using the phase difference between two sampled antenna signals. The data transmitted, which represents the specific drone’s propagated RF-signals, is processed by GNU-Radio Software, through which the computed AoA signal is transmitted over SPI to a control system tracking the drone. The antenna array, the correlating filters, the SDR, and the AoA is found to be feasible in simulation.
