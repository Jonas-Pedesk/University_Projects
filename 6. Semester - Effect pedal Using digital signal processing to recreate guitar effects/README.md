# Effect Pedal: Digital Signal Processing for Guitar Effects | Feb. 2024 – July 2024
The iconic rock sound of the electric guitars is a product of signal processing. This project aims to understand the fundamentals behind various digital audio effects commonly applied to electric guitar signals and implement three common effects: distortion, delay, and wah. The developed system mainly consists of an ESP32 MCU and a WM8960 Audio Codec. While the implementation of the effects is successful and there is general contentedness with the sound of the processed signal, there are still features that have not been implemented. Some of the missing features include a high-impedance input, a UI, the possibility of bypassing individual effects, bypassing the signal by analog means, and the construction of a complete prototype. While all the effects have been implemented and a functional test board has been developed, there are still points of improvement.
