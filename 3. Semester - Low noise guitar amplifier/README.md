# Low-Noise Guitar Amplifier | Sep. 2022 – Dec. 2022
For the project of this semester it has been chosen to work with the suggestion "HI-FI amplifier" with the variation "Guitar amplifier with equalizer/ distortion". In guitar amplifiers the signal can often absorb a lot of interference from the surroundings of the amplifier. And as the signal is amplified in the guitar amplifier, then this noise will be amplified along with the signal, causing the noise to be audible. When the system is amplified to the point of adding distortion to the signal, then the noise increases more than the signal itself, which causes the quality of a performance to be greatly reduced. As a solution to this problem a guitar amplifier will be designed, where parts in the guitar amplifier will be optimized with this noise in mind.
